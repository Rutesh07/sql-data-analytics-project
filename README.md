# 📊 SQL Data Analytics Project

Welcome to the **SQL Data Analytics Project** repository! 🚀  
This project demonstrates how SQL can be used to perform real-world business analytics, including time-series analysis, cumulative metrics, performance evaluation, data segmentation, and part-to-whole analysis.

It is designed as a **portfolio project** to showcase practical SQL skills and analytical thinking using structured datasets and business-driven scenarios.

---

## 📖 Project Overview

This repository contains a collection of SQL scripts that solve common analytical problems faced by data analysts and business intelligence professionals.

The project focuses on:

- **Change Over Time Analysis** – Identify trends and patterns across time periods  
- **Cumulative Metrics** – Calculate running totals and growth metrics  
- **Performance Analysis** – Measure KPIs such as revenue, sales, and customer performance  
- **Data Segmentation** – Analyze data by customer groups, regions, or product categories  
- **Part-to-Whole Analysis** – Understand how segments contribute to overall totals  

---

## 🧠 Key Concepts & Skills Demonstrated

- Advanced SQL queries  
- Joins and subqueries  
- Common Table Expressions (CTEs)  
- Window functions (SUM OVER, RANK, ROW_NUMBER, LAG, LEAD)  
- Aggregations and filtering  
- Analytical problem-solving  
- Business-focused data insights

## 🛠️ Tools & Technologies
